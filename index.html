Web based supply chain management system (User & Task management, Sales module, Production module, Stock module, Pricing module and Shipping module). Using Languages, Tools, Software and Platform: Classic ASP, VBScript, Java Script, CSS, MS SQL Server 2000, IIS, Windows 2003 Server. Third Party Tools: ABCpdf dynamically Adobe PDF reporting component, AspEmail is an active server component for sending e-mail messages using an external SMTP server in an ASP or VB environment component, AspJpeg is an active server component that creates resized versions of JPEG images component, AspUpload component. Responsibility: Team leader / Design, development and implementation. Major Parts in Responsibility: Customer dealings via email, Teamviewer, Skype or over telephone, requirement management, change request management, estimation, task scheduling, monitoring
